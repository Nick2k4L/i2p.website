## Forums {#forums}

Veuillez visiter notre forum utilisateur I2P - i2pforum.i2p (disponible sur l'internet non-privé à l'adresse https://i2pforum.net.

La discussion sur le développement d'I2P se déroule sur le forum utilisateur I2P. C'est généralement le meilleur endroit pour commencer avec les demandes de développeurs, si le canal IRC de développement est inactif. Pour le support général, utilisez soit le forum utilisateur I2P soit le Subreddit.

## IRC {#irc}

Notre réseau IRC principal est le réseau Irc2P au sein d'I2P ; un tunnel par défaut vers ce réseau est configuré lors des nouvelles installations de router.

**Liste des canaux :**

```
#i2p       - General i2p discussion and help channel
#i2p-chat  - Offtopic
#i2p-dev   - Development talk
```
<a href="https://irc.i2p.net" class="btn btn-primary" target="_blank" rel="noopener">Client IRC Web (Bêta)</a>

<small>Connectez-vous à I2P IRC depuis le clearnet. 100% sans JavaScript, alimenté par l'<a href="https://github.com/go-i2p/go-i2p" target="_blank" rel="noopener">API Go SAMv3</a> sécurisée.</small>

## Inclusion

I2P accueille toutes sortes de personnes, tant qu'elles sont amicales et s'entraident.

Le logiciel I2P router a été créé dans cet esprit et devrait être utilisé pour aider les personnes opprimées à retrouver leur liberté d'expression, sans pour autant opprimer les autres.
