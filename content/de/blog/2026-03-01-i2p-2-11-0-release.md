Ab dieser Version benötigt I2P nun Java 17 oder höher. Bitte konsultieren Sie die Dokumentation Ihres Betriebssystems, um zu erfahren, wie Sie Java aktualisieren können. I2P verwendet nun auch Jetty 12. Bitte überprüfen Sie Ihre router-Logs auf Migrationsprobleme.

Die Arbeit in dieser Version setzt die Verbesserung der Unterstützung für Post-Quantum-Kryptographie fort. Post-Quantum-Krypto ist jetzt standardmäßig auf der ratchet-Ebene aktiviert.

Wir haben die erste Runde von Gegenmaßnahmen gegen die laufenden Spam-Angriffe implementiert. Weitere Gegenmaßnahmen werden entwickelt.

Die SAMv3 API wurde erweitert, um die neuen Datagrammtypen 2 und 3 zu unterstützen.

Wie üblich empfehlen wir Ihnen, auf diese Version zu aktualisieren. Der beste Weg, die Sicherheit zu gewährleisten und dem Netzwerk zu helfen, ist die neueste Version zu verwenden.
