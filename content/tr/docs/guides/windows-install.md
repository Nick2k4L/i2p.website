## Easy Install Bundle

### ⚙️ Standart Kurulum

**I2P Kolay Kurulum Paketi**, Windows kullanıcıları için önerilen kurulum yöntemidir. Bu hepsi bir arada yükleyici, I2P'ye başlamak için ihtiyacınız olan her şeyi içerir:

- **I2P Router** - Temel I2P yazılımı
- **Gömülü Java Çalışma Ortamı** - Ayrı Java kurulumu gerektirmez
- **Firefox profilleri ve eklentileri** - Güvenli gezinme için I2P'ye optimize edilmiş tarayıcı profilleri ve eklentileri
- **Basit kurulum programı** - Manuel yapılandırma gerektirmez
- **Otomatik güncellemeler** - I2P yazılımınızı güncel tutun

Bu beta yükleyici, Java'yı doğrudan paketleyerek kurulum sürecini basitleştirir, böylece Java'yı ayrıca indirmenize veya yapılandırmanıza gerek kalmaz.

## Adım 3: Kurulum Klasörünü Seçin

Kesinlikle zorunlu olmasa da, **port yönlendirme I2P deneyiminizi önemli ölçüde iyileştirir** çünkü router'ınızın diğer I2P router'larıyla daha etkili iletişim kurmasını sağlar. Port yönlendirme olmadan I2P'yi kullanmaya devam edebilirsiniz, ancak performans ve ağa katkı azalır.

### Why Forward a Port?

- **Daha iyi bağlantı**: Diğer I2P router'larından gelen bağlantılara izin verir
- **Daha hızlı entegrasyon**: Ağa daha hızlı entegre olmanıza yardımcı olur
- **Ağa katkı**: Sizi I2P ağında daha iyi bir katılımcı yapar
- **Gelişmiş performans**: Genellikle daha iyi tunnel güvenilirliği ve hız sağlar

### Easy Install Bundle Nedir?

İlk olarak, I2P'nin hangi portu kullandığını belirlemeniz gerekir (varsayılan olarak rastgele atanır).

1. **I2P simgesini bulun** - Windows sistem tepsisinde (bildirim alanı) ekranınızın sağ alt köşesinde

![I2P Sistem Tepsisi Menüsü](/images/guides/windows-install/system-tray-menu.png)

2. **Bağlam menüsünü açmak için I2P simgesine sağ tıklayın**
3. **I2P router konsolunu açmak için "Launch I2P Browser"a tıklayın**

Menü ayrıca şu yararlı seçenekleri gösterir: - **Network: Firewalled** - Mevcut ağ durumunuzu gösterir - **Configure I2P System Tray** - Sistem tepsisi simgesi ayarlarını özelleştirin - **Stop I2P** / **Stop I2P Immediately** - Kapatma seçenekleri

### Finding Your Port Numbers

I2P tarayıcısı açıldıktan sonra, I2P'nin hangi portları kullandığını kontrol etmeniz gerekir:

1. **Ağ yapılandırma sayfasına gidin**:
   - Tarayıcınızda `http://127.0.0.1:7657/confignet` adresine gidin
   - Veya router konsol kenar çubuğundan: **Configuration → Network**

2. Port yapılandırma bölümüne **aşağı kaydırın**

![I2P Port Yapılandırması](/images/guides/windows-install/port-configuration.png)

3. **Gösterilen port numaralarını not edin**:

**UDP Yapılandırması**:    - **UDP portu**: Burada gösterilen port (örnek: `13697`)    - Varsayılan olarak, rastgele atanmış bir sayı ile "Belirli Bir Port Kullan" şeklinde ayarlanmıştır

**TCP Yapılandırması**: - **Dışarıdan erişilebilir TCP portu**: Genellikle UDP ile aynı portu kullanacak şekilde ayarlanır - Yukarıdaki örnekte: "UDP için yapılandırılan aynı portu kullan (şu anda 13697)"

**Önemli**: Yönlendiricinizde/güvenlik duvarınızda aynı port numarasında (bu örnekte, port `13697`) **hem UDP hem de TCP** yönlendirmesi yapmanız gerekir.

### How to Forward Your Port

Her router ve güvenlik duvarı farklı olduğundan, evrensel talimatlar sağlayamıyoruz. Ancak, [portforward.com](https://portforward.com/) binlerce router modeli için detaylı kılavuzlara sahiptir:

1. **[portforward.com](https://portforward.com/)** adresini ziyaret edin
2. Router (yönlendirici) üreticinizi ve modelinizi seçin
3. Port yönlendirmesi için adım adım kılavuzu takip edin
4. I2P yapılandırmanızda gösterilen port numarasında **hem UDP hem de TCP** protokollerini yönlendirin

**Genel adımlar** (router'a göre değişir): - Router'ınızın yönetici arayüzüne giriş yapın (genellikle `192.168.1.1` veya `192.168.0.1` adresinde) - "Port Forwarding" veya "Virtual Servers" bölümünü bulun - I2P port numaranız için yeni bir port forwarding kuralı oluşturun - Hem UDP hem de TCP protokollerini ayarlayın - Kuralı bilgisayarınızın yerel IP adresine yönlendirin - Yapılandırmayı kaydedin

Portunuzu yönlendirdikten sonra, I2P sistem tepsisi menüsünde "Network: Firewalled" durumundan "Network: OK" durumuna geçmelidir (bu birkaç dakika sürebilir).

## Sırada Ne Var

### What is the Standard Installation?

**Standart I2P kurulumu**, Windows üzerinde I2P kurmanın geleneksel yöntemidir. Easy Install Bundle'dan farklı olarak, bu yöntem şunları yapmanızı gerektirir:

- **Java'yı ayrı olarak kurun** - I2P'yi kurmadan önce Java Runtime Environment (JRE) indirip kurun
- **JAR kurulum dosyasını çalıştırın** - Java tabanlı grafiksel kurulum aracını kullanın
- **Manuel olarak yapılandırın** - Tarayıcı yapılandırmalarını kendiniz ayarlayın (isteğe bağlı)

Bu yöntem şunlar için önerilir: - Zaten Java yüklü olan kullanıcılar - Kurulum üzerinde daha fazla kontrol isteyen ileri düzey kullanıcılar - Geleneksel kurulum yöntemini tercih eden kullanıcılar - Easy Install Bundle'ın uyumlu olmadığı sistemler

### Java Requirements

- **Java Sürümü**: Java 8 (1.8) veya üzeri gereklidir
- **Önerilen**: Java 11 veya sonrası (LTS sürümü)
- **Tür**: Java Runtime Environment (JRE) veya Java Development Kit (JDK)

**Seçenek 2: OpenJDK** - Açık kaynaklı uygulama: [openjdk.org](https://openjdk.org) - Ücretsiz ve açık kaynaklı

**Seçenek 3: Adoptium (Eclipse Temurin)** - Önerilen alternatif: [adoptium.net](https://adoptium.net) - Ücretsiz, açık kaynaklı ve iyi bakımlı LTS sürümleri

**Java'nın kurulu olduğunu doğrulamak için**: 1. Komut İstemi'ni açın (`Windows + R` tuşlarına basın, `cmd` yazın, Enter'a basın) 2. Şunu yazın: `java -version` 3. Java sürümünüzü gösteren bir çıktı görmelisiniz

## Step 1: Install Java

I2P'yi kurmadan önce, sisteminize Java yüklemeniz gerekir.

1. **Bir Java dağıtımı seçin**:
   - **Oracle Java**: [java.com/download](https://www.java.com/download)

   - **Adoptium**: [adoptium.net](https://adoptium.net)

2. **Seçtiğiniz dağıtım için Windows yükleyicisini indirin**

3. **Yükleyiciyi çalıştırın** ve kurulum adımlarını takip edin

4. **Kurulumu doğrulayın**:
   - Komut İstemi'ni açın
   - `java -version` yazın ve Enter tuşuna basın

## Adım 3: Dilinizi Seçin

- **Entegrasyonu bekleyin**: I2P'nin ağa entegre olması ve tunnel'ları oluşturması için 5-10 dakika bekleyin
- **Port yönlendirmeyi yapılandırın**: Talimatlar için [port yönlendirme kılavuzuna](#recommended-port-forwarding-optional-but-important) bakın
- **Tarayıcınızı yapılandırın**: Web tarayıcınızı I2P'nin HTTP proxy'sini kullanacak şekilde ayarlayın
- **Router konsolunu keşfedin**: I2P'nin özellikleri, hizmetleri ve yapılandırma seçenekleri hakkında bilgi edinin
- **Eepsite'ları ziyaret edin**: I2P ağı üzerinden `.i2p` web sitelerine erişmeyi deneyin
- **Dokümantasyonu okuyun**: Daha fazla bilgi için [I2P dokümantasyonuna](/docs/) göz atın

I2P ağına hoş geldiniz! 🎉
