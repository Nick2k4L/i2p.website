## 论坛 {#forums}

请访问我们的 I2P 用户论坛 - i2pforum.i2p（在非私人网络上可通过 https://i2pforum.net 访问）。

关于I2P开发的讨论在I2P用户论坛上进行。如果开发者IRC频道不活跃，这通常是开始开发者咨询的最佳地点。对于一般支持，请使用I2P用户论坛或Subreddit。

## IRC {#irc}

我们的主要 IRC 网络是 I2P 内的 Irc2P 网络；新安装的 router 会默认设置到该网络的 tunnel。

**频道列表：**

```
#i2p       - General i2p discussion and help channel
#i2p-chat  - Offtopic
#i2p-dev   - Development talk
```
<a href="https://irc.i2p.net" class="btn btn-primary" target="_blank" rel="noopener">Web IRC 客户端 (Beta)</a>

<small>从明网连接到 I2P IRC。100% 无 JavaScript，由安全的 <a href="https://github.com/go-i2p/go-i2p" target="_blank" rel="noopener">Go SAMv3 API</a> 提供支持。</small>

## 包容性

I2P 欢迎所有类型的人，只要他们彼此友善和互相帮助。

I2P router 软件正是基于这种理念而创建的，应该用来帮助受压迫的人们重获言论自由，而不是压制他人。
