从此版本开始，I2P 现在需要 Java 17 或更高版本。请查阅您操作系统的文档以了解如何更新 Java。I2P 现在也使用 Jetty 12。请检查您的 router 日志以查看迁移问题。

本次版本的工作继续改进对后量子密码学的支持。后量子加密现在默认在棘轮层启用。

我们已经实施了针对持续垃圾信息攻击的第一轮缓解措施。正在开发更多的缓解措施。

SAMv3 API 已升级以支持新的数据报类型 2 和 3。

如往常一样，我们建议您升级到此版本。维护安全和帮助网络的最佳方式是运行最新版本。
