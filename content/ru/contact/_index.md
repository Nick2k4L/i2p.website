## Форумы {#forums}

Пожалуйста, посетите наш пользовательский форум I2P - i2pforum.i2p (доступен в обычном интернете по адресу https://i2pforum.net).

Обсуждение разработки I2P происходит на пользовательском форуме I2P. Обычно это лучшее место для начала обращений к разработчикам, если IRC-канал разработчиков неактивен. Для общей поддержки используйте либо пользовательский форум I2P, либо Subreddit.

## IRC {#irc}

Наша основная IRC-сеть — это сеть Irc2P внутри I2P; tunnel по умолчанию к этой сети настраивается при установке нового router.

**Список каналов:**

```
#i2p       - General i2p discussion and help channel
#i2p-chat  - Offtopic
#i2p-dev   - Development talk
```
<a href="https://irc.i2p.net" class="btn btn-primary" target="_blank" rel="noopener">Веб IRC-клиент (Бета)</a>

<small>Подключитесь к I2P IRC из clearnet. 100% без JavaScript, работает на основе безопасного <a href="https://github.com/go-i2p/go-i2p" target="_blank" rel="noopener">Go SAMv3 API</a>.</small>

## Включение

I2P приветствует людей всех типов, при условии, что они дружелюбны и готовы помогать друг другу.

Программное обеспечение I2P router было создано в этом духе и должно использоваться для помощи угнетенным людям в восстановлении их свободы слова, не подавляя при этом других.
