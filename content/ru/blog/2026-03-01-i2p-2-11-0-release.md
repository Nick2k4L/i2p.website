Начиная с этого релиза, I2P теперь требует Java 17 или более позднюю версию. Пожалуйста, обратитесь к документации вашей операционной системы для получения инструкций по обновлению Java. I2P теперь также использует Jetty 12. Пожалуйста, проверьте логи вашего router на предмет проблем с миграцией.

Работа в этом релизе продолжает улучшать поддержку постквантовой криптографии. Постквантовая криптография теперь включена по умолчанию на уровне ratchet.

Мы внедрили первый раунд мер по противодействию продолжающимся спам-атакам. Дополнительные меры защиты находятся в разработке.

SAMv3 API был обновлен для поддержки новых типов датаграмм 2 и 3.

Как обычно, мы рекомендуем обновиться до этой версии. Лучший способ поддерживать безопасность и помогать сети — использовать последнюю версию.
