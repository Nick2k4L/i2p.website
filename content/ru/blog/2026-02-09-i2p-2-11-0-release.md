Начиная с этой версии, I2P теперь требует Java 17 или более позднюю версию. Пожалуйста, обратитесь к документации вашей операционной системы для получения информации о том, как обновить Java. I2P теперь также использует Jetty 12. Пожалуйста, проверьте логи вашего router'а на наличие проблем с миграцией.

Работа в этом выпуске продолжает улучшать поддержку постквантовой криптографии. Постквантовая криптография теперь включена на уровне ratchet по умолчанию.

Мы внедрили первый этап мер противодействия продолжающимся спам-атакам. Разрабатываются дополнительные меры защиты.

SAMv3 API был обновлен для поддержки новых типов датаграмм 2 и 3.

Как обычно, мы рекомендуем обновиться до этой версии. Лучший способ поддерживать безопасность и помогать сети — использовать последнюю версию.
