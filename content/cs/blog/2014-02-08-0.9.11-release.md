0.9.11 pokračuje v zlepšování vyhledávání a ukládání LeaseSet, aby se zabránilo spojení Destination s Routerem útočníkem. Přidává podporu pro plugin [Orchid](http://www.subgraph.com/orchid.html) pro outproxy, který je dostupný na plugins.i2p. Díky opravám v transportech došlo ke snížení využití paměti. Dále obsahuje několik vylepšení protokolu I2CP, která poskytnou lepší možnosti vyhledávání a ochranu autorizace pro externí klienty. Samozřejmě jsou zde také běžné opravy chyb. Všichni uživatelé by měli aktualizovat.

Toto může být poslední verze, která bude fungovat s Java 5, která je již velmi stará a nepodporovaná. Pokud používáte Java 5 nebo 6, důrazně doporučujeme aktualizovat na Java 7.

Jak je zvykem, doporučujeme aktualizaci na tuto verzi. Nejlepší způsob, jak zajistit bezpečnost a pomoci síti, je používat nejnovější verzi.

## PODROBNOSTI VYDÁNÍ

### Vylepšení anonymity

- Další vylepšení zpracování leaseset

### Opravy chyb

- Oprava NPE po vypnutí klienta
- Oprava kódování logů wrapperu na stránce s logy
- Opravy pingování ve Streaming a I2Ping

### Ostatní

- Přidána podpora pro plugin Orchid
- Přidána podpora HTTPS pro HTTP klienta proxy
- Nová podpora I2CP pro vyhledávání hostname externími klienty
- Přísnější vynucování autorizace I2CP pro externí klienty (nepříznivá změna)
- Zvýšena výchozí variabilita délky příchozích průzkumných tunelů
- Výrazné snížení využití paměti transporty
- Všechny aktualizace v síti přes torrent
- Aktualizace překladů
- Aktualizace dat GeoIP (pouze nové instalace a PPA)
