La versión 0.7.7 de I2P contiene varias mejoras en anonimato. El uso de claves de sesión para la criptografía fue completamente reestructurado, y ahora I2PSnark utiliza sus propios túneles para la comunicación con el tracker. Continúa el trabajo para mejorar la consola del router, incluyendo el cumplimiento de los estándares HTML y un mejor soporte de UTF-8.

También continúa el trabajo para aumentar las velocidades, tanto reduciendo la cantidad de claves de sesión transferidas, como mejorando los métodos para descartar mensajes durante sobrecarga, y reduciendo los descartes por parte de routers de alto ancho de banda. Hay una nueva opción de actualización sin firmar para aquellos que deseen actualizaciones automáticas a versiones de desarrollo más recientes.

La versión también incluye varios cambios para reducir el uso de memoria y CPU, y una gran variedad de correcciones de errores. Se recomienda actualizar. Por favor, dé a los desarrolladores sus comentarios en IRC #i2p o en [forum.i2p2.de](http://forum.i2p2.de) y [participe](http://www.i2p2.de/getinvolved.html)!

## Detalles de la versión:

### Recursos

- Rediseño de filtros Bloom, ahorra 6-8MB de heap.
- Reducción de buffers PRNG para I2PAppContext, ahorra 3.5MB de heap.
- Eliminación de más clases y código no utilizados, más limpiezas para Java 5
- Reducción ligera de los límites de conexión.

### Túneles

- Mejora del descarte aleatorio temprano (random early discarder), para hacerlo más inteligente - esperamos que mejore las velocidades de los túneles
- Aumento del tamaño del filtro Bloom para routers de alto ancho de banda (>= 512KBps compartidos) para reducir descartes de paquetes

### Criptografía

- Uso de múltiples gestores de claves de sesión (Session Key Managers) para beneficios de anonimato
- Reducción del número de etiquetas de sesión (Session Tags) enviadas a la vez, para mayor eficiencia - debería reducir pausas en conexiones de larga duración

### Consola

- Muchas limpiezas y correcciones de HTML
- Correcciones de UTF-8
- Nueva página de error

### Actualizaciones

- Nueva opción de actualización sin firmar
- Verificación de la integridad del archivo zip al inicio

### I2PSnark

- Uso de los túneles de snark en lugar del proxy HTTP para anuncios y descargas de torrents, para mayor eficiencia y anonimato.
