date: 2026-02-09

categories: ["release"]
---

A partir de esta versión, I2P ahora requiere Java 17 o posterior. Por favor consulta la documentación de tu sistema operativo para saber cómo actualizar Java. I2P ahora también usa Jetty 12. Por favor revisa los logs de tu router para detectar problemas de migración.

El trabajo en esta versión continúa mejorando el soporte para criptografía post-cuántica. La criptografía post-cuántica ahora está habilitada por defecto en la capa ratchet.

Hemos implementado la primera ronda de mitigaciones para los ataques de spam en curso. Se están desarrollando mitigaciones adicionales.

La API SAMv3 ha sido actualizada para soportar los nuevos tipos de datagram 2 y 3.

Como siempre, recomendamos que actualices a esta versión. La mejor manera de mantener la seguridad y ayudar a la red es ejecutar la versión más reciente.
