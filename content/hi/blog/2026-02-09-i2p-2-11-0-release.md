इस रिलीज़ के अनुसार, I2P को अब Java 17 या उससे नए वर्जन की आवश्यकता है। Java को अपडेट करने के लिए कृपया अपने ऑपरेटिंग सिस्टम के दस्तावेज़ देखें। I2P अब Jetty 12 का भी उपयोग करता है। कृपया माइग्रेशन संबंधी समस्याओं के लिए अपने router लॉग्स जांचें।

इस रिलीज़ में काम post-quantum cryptography के लिए समर्थन में सुधार जारी रखता है। Post-Quantum crypto अब ratchet layer पर डिफ़ॉल्ट रूप से सक्षम है।

हमने चल रहे स्पैम हमलों के लिए शमन उपायों का पहला चरण लागू किया है। अतिरिक्त शमन उपाय विकसित किए जा रहे हैं।

SAMv3 API को नए datagram प्रकार 2 और 3 का समर्थन करने के लिए उन्नत किया गया है।

हमेशा की तरह, हम सुझाते हैं कि आप इस रिलीज़ में अपडेट करें। सुरक्षा बनाए रखने और नेटवर्क की मदद करने का सबसे अच्छा तरीका है नवीनतम रिलीज़ चलाना।
