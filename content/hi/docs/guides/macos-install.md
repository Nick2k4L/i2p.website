## आपको क्या चाहिए होगा

- macOS 10.14 (Mojave) या उसके बाद का वर्जन चलाने वाला Mac
- एप्लिकेशन इंस्टॉल करने के लिए Administrator पहुंच
- लगभग 15-20 मिनट का समय
- इंस्टॉलर डाउनलोड करने के लिए इंटरनेट कनेक्शन

## अवलोकन

इस इंस्टॉलेशन प्रक्रिया के चार मुख्य चरण हैं:

1. **Java स्थापित करें** - Oracle Java Runtime Environment डाउनलोड और स्थापित करें
2. **I2P स्थापित करें** - I2P installer डाउनलोड करें और चलाएं
3. **I2P App कॉन्फ़िगर करें** - launcher सेट करें और अपने dock में जोड़ें
4. **I2P Bandwidth कॉन्फ़िगर करें** - अपने कनेक्शन को अनुकूलित करने के लिए setup wizard चलाएं

## भाग एक: Java इंस्टॉल करें

I2P को चलाने के लिए Java की आवश्यकता होती है। यदि आपके पास पहले से Java 8 या उसके बाद का संस्करण स्थापित है, तो आप [भाग दो पर जा सकते हैं](#part-two-download-and-install-i2p)।

### चरण 1: Java डाउनलोड करें

[Oracle Java download page](https://www.oracle.com/java/technologies/downloads/) पर जाएं और Java 8 या इसके बाद के संस्करण के लिए macOS installer डाउनलोड करें।

![macOS के लिए Oracle Java डाउनलोड करें](/images/guides/macos-install/0-jre.png)

### चरण 2: इंस्टॉलर चलाएं

अपने Downloads फ़ोल्डर में डाउनलोड की गई `.dmg` फ़ाइल को खोजें और इसे खोलने के लिए डबल-क्लिक करें।

![Java installer खोलें](/images/guides/macos-install/1-jre.png)

### चरण 3: इंस्टॉलेशन की अनुमति दें

macOS एक सुरक्षा प्रॉम्प्ट दिखा सकता है क्योंकि इंस्टॉलर एक पहचाने गए डेवलपर से है। आगे बढ़ने के लिए **Open** पर क्लिक करें।

![इंस्टॉलर को आगे बढ़ने की अनुमति दें](/images/guides/macos-install/2-jre.png)

### चरण 4: Java स्थापित करें

Java इंस्टॉलेशन प्रक्रिया शुरू करने के लिए **Install** पर क्लिक करें।

![Java स्थापित करना शुरू करें](/images/guides/macos-install/3-jre.png)

### चरण 5: स्थापना की प्रतीक्षा करें

इंस्टॉलर आपके सिस्टम पर फाइलें कॉपी करेगा और Java को कॉन्फ़िगर करेगा। इसमें आमतौर पर 1-2 मिनट लगते हैं।

![इंस्टॉलर के पूरा होने का इंतजार करें](/images/guides/macos-install/4-jre.png)

### चरण 6: स्थापना पूर्ण

जब आप सफलता का संदेश देखें, तो Java इंस्टॉल हो गया है! समाप्त करने के लिए **Close** पर क्लिक करें।

![Java installation complete](/images/guides/macos-install/5-jre.png)

## भाग दो: I2P डाउनलोड और इंस्टॉल करें

अब जब Java इंस्टॉल हो गया है, आप I2P router इंस्टॉल कर सकते हैं।

### चरण 1: I2P डाउनलोड करें

[डाउनलोड पेज](/downloads/) पर जाएं और **I2P for Unix/Linux/BSD/Solaris** इंस्टॉलर (`.jar` फ़ाइल) डाउनलोड करें।

![Download I2P installer](/images/guides/macos-install/0-i2p.png)

### चरण 2: इंस्टॉलर चलाएं

डाउनलोड की गई `i2pinstall_X.X.X.jar` फ़ाइल पर डबल-क्लिक करें। installer लॉन्च हो जाएगा और आपसे अपनी पसंदीदा भाषा चुनने को कहेगा।

![अपनी भाषा चुनें](/images/guides/macos-install/1-i2p.png)

### चरण 3: स्वागत स्क्रीन

स्वागत संदेश पढ़ें और जारी रखने के लिए **Next** पर क्लिक करें।

![Installer introduction](/images/guides/macos-install/2-i2p.png)

### चरण 4: महत्वपूर्ण सूचना

इंस्टॉलर अपडेट के बारे में एक महत्वपूर्ण सूचना प्रदर्शित करेगा। I2P अपडेट **end-to-end signed** और सत्यापित हैं, भले ही यह इंस्टॉलर स्वयं unsigned है। **Next** पर क्लिक करें।

![अपडेट के बारे में महत्वपूर्ण सूचना](/images/guides/macos-install/3-i2p.png)

### चरण 5: लाइसेंस समझौता

I2P लाइसेंस समझौता (BSD-स्टाइल लाइसेंस) पढ़ें। स्वीकार करने के लिए **Next** पर क्लिक करें।

![License agreement](/images/guides/macos-install/4-i2p.png)

### चरण 6: इंस्टॉलेशन डायरेक्टरी चुनें

I2P को कहाँ इंस्टॉल करना है चुनें। डिफ़ॉल्ट स्थान (`/Applications/i2p`) की सिफारिश की जाती है। **Next** पर क्लिक करें।

![इंस्टॉलेशन डायरेक्टरी चुनें](/images/guides/macos-install/5-i2p.png)

### चरण 7: घटकों का चयन करें

पूर्ण स्थापना के लिए सभी घटकों को चयनित रखें। **Next** पर क्लिक करें।

![इंस्टॉल करने के लिए घटकों का चयन करें](/images/guides/macos-install/6-i2p.png)

### चरण 8: इंस्टॉलेशन शुरू करें

अपने विकल्पों की समीक्षा करें और I2P इंस्टॉल करना शुरू करने के लिए **Next** पर क्लिक करें।

![इंस्टॉलेशन शुरू करें](/images/guides/macos-install/7-i2p.png)

### चरण 9: फाइलें स्थापित करना

इंस्टॉलर आपके सिस्टम में I2P फाइलों की प्रतिलिपि बनाएगा। इसमें लगभग 1-2 मिनट लगते हैं।

![इंस्टॉलेशन प्रगति में](/images/guides/macos-install/8-i2p.png)

### चरण 10: लॉन्च स्क्रिप्ट्स जेनरेट करें

इंस्टॉलर I2P शुरू करने के लिए लॉन्च स्क्रिप्ट बनाता है।

![लॉन्च स्क्रिप्ट तैयार करना](/images/guides/macos-install/9-i2p.png)

### चरण 11: इंस्टॉलेशन शॉर्टकट

इंस्टॉलर डेस्कटॉप शॉर्टकट और मेन्यू एंट्रीज़ बनाने का विकल्प देता है। अपने चयन करें और **Next** पर क्लिक करें।

![शॉर्टकट बनाएं](/images/guides/macos-install/10-i2p.png)

### चरण 12: इंस्टॉलेशन पूर्ण

सफलता! I2P अब इंस्टॉल हो गया है। समाप्त करने के लिए **Done** पर क्लिक करें।

![स्थापना पूर्ण](/images/guides/macos-install/11-i2p.png)

## भाग तीन: I2P ऐप को कॉन्फ़िगर करें

अब आइए I2P को आसानी से लॉन्च करने के लिए इसे अपने Applications फ़ोल्डर और Dock में जोड़ते हैं।

### चरण 1: एप्लिकेशन फ़ोल्डर खोलें

Finder खोलें और अपने **Applications** फ़ोल्डर पर जाएं।

![Applications फ़ोल्डर खोलें](/images/guides/macos-install/0-conf.png)

### चरण 2: I2P Launcher खोजें

`/Applications/i2p/` के अंदर **I2P** फ़ोल्डर या **Start I2P Router** एप्लिकेशन को खोजें।

![I2P launcher खोजें](/images/guides/macos-install/1-conf.png)

### चरण 3: Dock में जोड़ें

आसान पहुंच के लिए **Start I2P Router** एप्लिकेशन को अपने Dock पर खींचें। आप अपने desktop पर एक alias भी बना सकते हैं।

![अपने Dock में I2P जोड़ें](/images/guides/macos-install/2-conf.png)

**सुझाव**: Dock में I2P आइकन पर राइट-क्लिक करें और **Options → Keep in Dock** चुनें ताकि इसे स्थायी रूप से रखा जा सके।

## भाग चार: I2P बैंडविड्थ कॉन्फ़िगर करें

जब आप पहली बार I2P लॉन्च करते हैं, तो आप अपनी bandwidth सेटिंग्स को configure करने के लिए एक setup wizard चलाएंगे। यह आपके कनेक्शन के लिए I2P के प्रदर्शन को अनुकूलित करने में मदद करता है।

### चरण 1: I2P लॉन्च करें

अपने Dock में I2P आइकन पर क्लिक करें (या launcher को डबल-क्लिक करें)। आपका डिफ़ॉल्ट वेब ब्राउज़र I2P Router Console पर खुल जाएगा।

![I2P Router Console स्वागत स्क्रीन](/images/guides/macos-install/0-wiz.png)

### चरण 2: स्वागत विज़ार्ड

सेटअप विज़ार्ड आपका स्वागत करेगा। I2P को कॉन्फ़िगर करना शुरू करने के लिए **Next** पर क्लिक करें।

![Setup wizard introduction](/images/guides/macos-install/1-wiz.png)

### चरण 3: भाषा और थीम

अपनी पसंदीदा **interface language** चुनें और **light** या **dark** theme के बीच चयन करें। **Next** पर क्लिक करें।

![भाषा और थीम चुनें](/images/guides/macos-install/2-wiz.png)

### चरण 4: बैंडविड्थ परीक्षण जानकारी

विज़ार्ड बैंडविड्थ टेस्ट की व्याख्या करेगा। यह टेस्ट आपकी इंटरनेट स्पीड मापने के लिए **M-Lab** सेवा से कनेक्ट करता है। आगे बढ़ने के लिए **Next** पर क्लिक करें।

![Bandwidth test explanation](/images/guides/macos-install/3-wiz.png)

### चरण 5: बैंडविड्थ टेस्ट चलाएं

अपलोड और डाउनलोड स्पीड मापने के लिए **Run Test** पर क्लिक करें। टेस्ट में लगभग 30-60 सेकंड का समय लगता है।

![बैंडविड्थ परीक्षण चलाना](/images/guides/macos-install/4-wiz.png)

### चरण 6: परीक्षण परिणाम

अपने परीक्षण परिणामों की समीक्षा करें। I2P आपकी कनेक्शन गति के आधार पर बैंडविड्थ सेटिंग्स की सिफारिश करेगा।

![Bandwidth test results](/images/guides/macos-install/5-wiz.png)

### चरण 7: बैंडविड्थ साझाकरण कॉन्फ़िगर करें

चुनें कि आप I2P नेटवर्क के साथ कितनी बैंडविड्थ साझा करना चाहते हैं:

- **स्वचालित** (अनुशंसित): I2P आपके उपयोग के आधार पर bandwidth का प्रबंधन करता है
- **सीमित**: विशिष्ट upload/download सीमाएं निर्धारित करें
- **असीमित**: जितना संभव हो उतना साझा करें (तेज़ कनेक्शन के लिए)

अपनी सेटिंग्स सेव करने के लिए **Next** पर क्लिक करें।

![बैंडविड्थ साझाकरण कॉन्फ़िगर करें](/images/guides/macos-install/6-wiz.png)

### चरण 8: कॉन्फ़िगरेशन पूर्ण

आपका I2P router अब कॉन्फ़िगर और चालू है! Router console आपकी कनेक्शन स्थिति दिखाएगा और आपको I2P साइटों को ब्राउज़ करने की अनुमति देगा।

## I2P के साथ शुरुआत करना

अब जब I2P स्थापित और कॉन्फ़िगर हो गया है, आप कर सकते हैं:

1. **I2P साइटों को ब्राउज़ करें**: लोकप्रिय I2P सेवाओं के लिंक देखने हेतु [I2P होमपेज](http://127.0.0.1:7657/home) पर जाएं
2. **अपना ब्राउज़र कॉन्फ़िगर करें**: `.i2p` साइटों तक पहुंचने के लिए एक [ब्राउज़र प्रोफ़ाइल](/docs/guides/browser-config) सेट करें
3. **सेवाओं का अन्वेषण करें**: I2P ईमेल, फ़ोरम, फ़ाइल शेयरिंग, और अन्य की जांच करें
4. **अपने router की निगरानी करें**: [कंसोल](http://127.0.0.1:7657/console) आपकी नेटवर्क स्थिति और आंकड़े दिखाता है

### उपयोगी लिंक

- **Router Console**: [http://127.0.0.1:7657/](http://127.0.0.1:7657/)
- **कॉन्फ़िगरेशन**: [http://127.0.0.1:7657/config](http://127.0.0.1:7657/config)
- **पता पुस्तिका**: [http://127.0.0.1:7657/susidns/addressbook](http://127.0.0.1:7657/susidns/addressbook)
- **बैंडविड्थ सेटिंग्स**: [http://127.0.0.1:7657/config](http://127.0.0.1:7657/config)

## सेटअप विज़ार्ड को फिर से चलाना

यदि आप अपनी bandwidth settings को बदलना चाहते हैं या बाद में I2P को फिर से configure करना चाहते हैं, तो आप Router Console से welcome wizard को फिर से चला सकते हैं:

1. [I2P Setup Wizard](http://127.0.0.1:7657/welcome) पर जाएं
2. wizard के चरणों का फिर से पालन करें

## समस्या निवारण

### I2P शुरू नहीं हो रहा

- **Java जांचें**: Terminal में `java -version` चलाकर सुनिश्चित करें कि Java इंस्टॉल है
- **अनुमतियां जांचें**: सुनिश्चित करें कि I2P फ़ोल्डर में सही अनुमतियां हैं
- **लॉग जांचें**: त्रुटि संदेशों के लिए `~/.i2p/wrapper.log` देखें

### ब्राउज़र I2P साइटों तक पहुँच नहीं सकता

- सुनिश्चित करें कि I2P चल रहा है (Router Console चेक करें)
- अपने ब्राउज़र की proxy सेटिंग्स को HTTP proxy `127.0.0.1:4444` उपयोग करने के लिए कॉन्फ़िगर करें
- नेटवर्क में I2P के integrate होने के लिए शुरू करने के बाद 5-10 मिनट प्रतीक्षा करें

### धीमी गति

- bandwidth परीक्षण फिर से चलाएं और अपनी सेटिंग्स को समायोजित करें
- सुनिश्चित करें कि आप network के साथ कुछ bandwidth साझा कर रहे हैं
- Router Console में अपनी कनेक्शन स्थिति जांचें

## I2P को अनइंस्टॉल करना

अपने Mac से I2P को हटाने के लिए:

1. यदि I2P router चल रहा है तो इसे बंद करें
2. `/Applications/i2p` फ़ोल्डर को हटाएं
3. `~/.i2p` फ़ोल्डर को हटाएं (आपका I2P कॉन्फ़िगरेशन और डेटा)
4. अपने Dock से I2P आइकन को हटाएं
