A partir desta versão, o I2P agora requer Java 17 ou posterior. Por favor, consulte a documentação do seu sistema operacional sobre como atualizar o Java. O I2P agora também usa Jetty 12. Por favor, verifique os logs do seu router para problemas de migração.

O trabalho nesta versão continua a melhorar o suporte para criptografia pós-quântica. A criptografia pós-quântica agora está habilitada por padrão na camada ratchet.

Implementamos a primeira rodada de mitigações aos ataques de spam em andamento. Mitigações adicionais estão sendo desenvolvidas.

A API SAMv3 foi atualizada para suportar os novos tipos de datagrama 2 e 3.

Como de costume, recomendamos que você atualize para esta versão. A melhor forma de manter a segurança e ajudar a rede é executar a versão mais recente.
