## IRC {#irc}

Nossa rede IRC principal é a rede Irc2P dentro do I2P; um tunnel padrão para esta rede é configurado com novas instalações do router.

**Lista de canais:**

```
#i2p       - General i2p discussion and help channel
#i2p-chat  - Offtopic
#i2p-dev   - Development talk
```
<a href="https://irc.i2p.net" class="btn btn-primary" target="_blank" rel="noopener">Cliente IRC Web (Beta)</a>

<small>Conecte-se ao IRC do I2P pela clearnet. 100% livre de JavaScript, alimentado pela <a href="https://github.com/go-i2p/go-i2p" target="_blank" rel="noopener">API Go SAMv3</a> segura.</small>

## Inclusão

O I2P acolhe todos os tipos de pessoas, desde que sejam amigáveis e prestativas umas com as outras.

O software do router I2P foi criado neste espírito e deve ser usado para ajudar pessoas reprimidas a recuperarem sua liberdade de expressão, sem suprimir outras pessoas.
